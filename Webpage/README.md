# ⚡ Web Application (React + TypeScript + Vite)

A modern, fast web application component built with **React**, **TypeScript**, and **Vite**, along with an archived static prototype.

---

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vite.dev/)
- **Linter**: [Oxlint](https://oxc.rs/)

---

## 📁 Directory Overview

```text
Webpage/
├── src/                    # React source files (components, styles, assets)
│   ├── App.tsx             # Main Application component
│   ├── App.css             # Component-level styles
│   ├── index.css           # Global design system & base styling
│   └── main.tsx            # Application entry point
├── public/                 # Static public assets (icons, images)
├── legacy-static/          # Archived static HTML/CSS/JS prototype & target references
├── package.json            # Project dependencies and scripts
└── vite.config.ts          # Vite configuration
```

---

## 🚀 Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

### 4. Preview Production Build
```bash
npm run preview
```

---

## 🏛️ Legacy Static Prototype

The `legacy-static/` directory contains early HTML5/CSS3 static prototypes and initial design reference screenshots used during the project's ideation phase.
